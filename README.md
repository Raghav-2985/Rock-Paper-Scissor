# Rock-Paper-Scissor

This game is an exercise given in [The Odin Project](https://www.theodinproject.com/) curlicullum.

# How to download

You can dowload this game by cloning it locally using git bash : 
``` bash
git clone https://github.com/Raghav-2985/Rock-Paper-Scissor.git
```

# How to play

This game is completely terminal based so all the result will be shown at the console of your browser here is how to open it:

1. Right click anywhere on the page

2. You will see an option called "inspect" click on it

3. Click on the console at the top and there you go !!
